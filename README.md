# lorna-debugger
Heavy Metal Debugging with lorna-debug. Just another simple debugger written in Rust. 
- CURRENTLY:
Implementing easy to use Web Interface and some trees like in GHIDRA and IDA

## How to build

Build with:

    cargo build

## How to use

    ./lornaDebug ./EXEC
